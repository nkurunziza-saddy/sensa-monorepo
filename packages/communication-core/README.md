# @sensa-monorepo/communication-core

Internal core logic for Sensa communication modalities.

## ⚠️ Internal Only

This package is marked as `private: true` to prevent publication to npm.

It should only be used as a workspace dependency within the Sensa monorepo. It contains core logic that is shared between different communication-related packages and applications.

## Features

- Gesture mappings and translations.
- Hand landmark detection logic (pure functions).
- Shared communication types.
