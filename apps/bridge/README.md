# Sensa Bridge

A high-fidelity communication bridge application demonstrating the "Clay Refined" design system and the `@sensa-monorepo/communication` package.

## Features

- **Gesture Bridge**: Real-time sign-to-text conversion.
- **Speech Stream**: Hands-free voice-to-text communication.
- **Ultra-Minimal UI**: Focused on single-modality clarity.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS with "Clay Refined" design system.
- **Communication**: `@sensa-monorepo/communication`

## Getting Started

```bash
# Run from the root
turbo dev --filter bridge
```

The app will be available at `http://localhost:3002`.
